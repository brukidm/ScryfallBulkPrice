# ScryfallBulkPrice
Python script that calculates total price of MTG card list based on Scryfall trends

Requirements:
- requests
- re
- json

Sample input:
```
1 Crypt Rats
4 Dusk Legion Zealot
3 Gray Merchant
3 Stinkweed 
2 Defile
2 Choking Sands
2 Dauthi Slayer
2 Okiba-Gang Shinobi
3 Shred Memory
2 Shrivel
4 Wrench Mind
```
